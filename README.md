## ⚡ Quick Start Scripts (One-Click Helpers)

For a seamless local experience, we've bundled several Windows helper scripts directly in the root directory:

### 1️⃣ Launch Locally — `run-portfolio.bat`
Double-click this file to free up your ports, install all node dependencies, and run the Vite local server automatically.
* Opens at: **http://localhost:5173**

### 2️⃣ Diagnostics & Logs — `run-diagnostics.bat`
If anything goes wrong or the website doesn't load, double-click this file. It checks your local Node/NPM versions, runs the installer, does a quick server startup test, and outputs the detailed results to `diagnostics.log`.

### 3️⃣ Direct GitHub Deployer — `upload-to-github.bat`
Ready to share your code? Double-click this helper script, input your new GitHub repo URL, and it will run the initial Git initialization, staging, commits, and pushes for you automatically!

## 📬 Contact Form Mechanism (Web3Forms)
The contact form uses a modern serverless pipeline powered by **[Web3Forms](https://web3forms.com)**, forwarding submissions directly to your email without needing an active back-end server.

To activate:
1. Obtain a free access key at [web3forms.com](https://web3forms.com) (takes 10 seconds).
2. Create/edit the `.env` file in the project subfolder:
   ```env
   VITE_WEB3FORMS_KEY=your_copied_access_key
   ```
3. Commit and run the application!
